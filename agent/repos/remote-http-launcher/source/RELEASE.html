Release remote-http-launcher to PyPI

One-time setup

  • Ensure you have PyPI maintainer access for remote-http-launcher.
  • Install build tooling:
python -m pip install --upgrade build twine

Build

python -m build

Validate

python -m twine check dist/*

Upload

Test PyPI:

python -m twine upload --repository testpypi dist/*

Production PyPI:

python -m twine upload dist/*
