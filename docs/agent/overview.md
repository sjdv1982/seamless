# Agent docs overview

This page summarizes the scope of the new Seamless agent documentation.

## Scope

- Contract pages under `contracts/` define expected behavior.
- API pages under `api/` document the callable surface.
- `index.json` and `public-api.json` provide machine-readable indices.

## Legacy docs

Legacy Seamless (0.x) documentation remains available at `/legacy/` with a deprecation banner.
